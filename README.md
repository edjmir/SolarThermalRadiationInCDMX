# SolarThermalRadiationInCDMX

## Data source

These data were obtained from the NASA Langley Research Center (LaRC) POWER Project funded through the NASA Earth Science/Applied Science Program.

## Location and Timeline

To change location and time line please refer to the constants below

```
const LATITUDE::Float64 = (-)XX.XXXX;
const LONGITUDE::Float64 = (-)XX.XXXX;
const START_DATE::Int64 = YYYYMMDD;
const END_DATE::Int64 = YYYYMMDD;
```
